【LeetCode每日一题】1129. 颜色交替的最短路径

2023-02-02
724- 3m-
LeetCode每日一题-LeetCodeAlgorithm

题目链接：https://leetcode.cn/problems/shortest-path-with-alternating-colors/

题目难度：中等

题意

在一个有向图中，节点分别标记为 0, 1, …, n-1。图中每条边为红色或者蓝色，且存在自环或平行边。

red_edges 中的每一个 [i, j] 对表示从节点 i 到节点 j 的红色有向边。类似地，blue_edges 中的每一个 [i, j] 对表示从节点 i 到节点 j 的蓝色有向边。

返回长度为 n 的数组 answer，其中 answer[X] 是从节点 0 到节点 X 的红色边和蓝色边交替出现的最短路径的长度。如果不存在这样的路径，那么 answer[x] = -1。

示例1：

输入：n = 3, red_edges = [[0,1],[1,2]], blue_edges = []
输出：[0,1,-1]

示例2：

输入：n = 3, red_edges = [[0,1]], blue_edges = [[2,1]]
输出：[0,1,-1]

示例3：

输入：n = 3, red_edges = [[1,0]], blue_edges = [[2,1]]
输出：[0,-1,-1]

示例 4：

输入：n = 3, red_edges = [[0,1]], blue_edges = [[1,2]]
输出：[0,1,2]

提示：

  • $1 <= n <= 100$
  • $red_edges.length <= 400$
  • $blue_edges.length <= 400$
  • $red_edges[i].length == blue_edges[i].length == 2$
  • $0 <= red_edges[i][j], blue_edges[i][j] < n$

题解：BFS + 建图

首先建图，将红色边和蓝色边分成两种类型，红色有0表示，蓝色用1表示，则节点i到节点j的红蓝色边交替出现的路径为：

  • 类型0：路径最终到节点j的边为红色
  • 类型1：路径最终到节点j的边为蓝色

为了方便建图，设图数据结构为$edge[2][n][]$，即：

  • $edge[0][i][j]$：节点i和节点j之间的有向边为红色
  • $edge[1][i][j]$：节点i和节点j之间的有向边为蓝色

随后定义最短路径长度数组，为了方便处理，定义数据结构为$dis[2][n]$，即：

  • $dis[0][i]$：节点0到节点i，且最终到节点i的边为红色的最短路径长度
  • $dis[1][i]$：节点0到节点i，且最终到节点i的边为蓝色的最短路径长度

队列数据结构为：$queue> q$，即：存储下一个节点编号及其边的颜色

最终取$min(dis[0][i], dis[1][i])$即可。

C++代码

class Solution {
public:
    vector<int> shortestAlternatingPaths(int n, vector<vector<int>>& redEdges, vector<vector<int>>& blueEdges) {
        //建图
        vector< vector< vector<int> > > edge(2,vector< vector<int> >(n));
        for(auto &e : redEdges) {
            edge[0][e[0]].push_back(e[1]);
        }
        for(auto &e : blueEdges) {
            edge[1][e[0]].push_back(e[1]);
        }

        //bfs
        vector< vector<int> > dis(2, vector<int>(n,INT_MAX)); //两种颜色的最短路
        queue<pair<int,int>> q;  //记录当前节点及其边的颜色
        dis[0][0]=0, dis[1][0]=0; 
        q.push({0,0}), q.push({0,1});
        while(!q.empty()) {
            auto [u,t] = q.front(); q.pop();
            for(auto v : edge[1-t][u]) {
                if(dis[1-t][v] != INT_MAX) continue;
                dis[1-t][v] = dis[t][u] + 1;
                q.push({v,1-t});
            }
        }
        
        //ans
        vector<int> ans(n);
        for(int i=0; i<n; i++) {
            ans[i] = min(dis[0][i], dis[1][i]);
            if(ans[i] == INT_MAX) ans[i]=-1;
        }
        return ans;
    }
};
